# DWPK Caddy

Custom docker image of DWPK's web server.

[![Build and Publish Container Image](https://github.com/starfishpatkhoo/dwpk_caddy/actions/workflows/docker-publish.yaml/badge.svg)](https://github.com/starfishpatkhoo/dwpk_caddy/actions/workflows/docker-publish.yaml)

- FrankenPHP (Caddy) latest
- PHP 8.5.x
- Trixie (Debian v13)
- Cloudflare SSL Services Module
- Built manually on demand - see [container tags](https://github.com/starfishpatkhoo/dwpk_caddy/pkgs/container/dwpk_caddy/versions) for the build info/version

Pull this with `ghcr.io/starfishpatkhoo/dwpk_caddy:latest`.
